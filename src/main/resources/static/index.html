Swagger UI Raw OpenAPI JSON

Available Endpoints

  • GET /fnol/health - Service health
  • POST /fnol/email - Acknowledge email payload
  • GET /s3/health - S3 connectivity health
  • POST /s3/put - Upload file (multipart)
  • POST /s3/put-file-public - Upload bytes to public bucket (JSON)
  • POST /s3/put-bytes-public - Upload bytes to public bucket (JSON)
  • POST /s3/put-bytes - Upload bytes (JSON)
  • GET /s3/get - Download object
  • GET /s3/get-url - Get object URL
  • GET /s3/list - List objects

Notes

  • Swagger UI is available at /swagger-ui/index.html.
  • OpenAPI JSON is served at /v3/api-docs.
  • For S3 endpoints, credentials and region should be provided via environment variables on your deployment platform.
